# Информационно-справочная система интернет-магазина

Курсовой проект по дисциплине «Современные технологии программирования».

## Стек технологий

- Java 17  
- Spring Boot 3 (Web, Security, Data JPA, Validation, Thymeleaf)  
- PostgreSQL  
- Maven

## Как запустить

1. Ничего дополнительно устанавливать не нужно: по умолчанию приложение использует встроенную базу H2 (в памяти).
   Консоль H2 будет доступна по адресу `http://localhost:8080/h2-console` после запуска.
2. Если требуется подключить внешнюю PostgreSQL, укажите параметры подключения в файле
   `src/main/resources/application.properties` (по умолчанию используются значения для H2).
3. В корне проекта выполнить команду:
   ```bash
   mvn spring-boot:run
   ```
   При первом запуске Hibernate автоматически создаст все необходимые таблицы
   и инициализирует тестовые данные (администратор, категории, товары).
4. Если внешние репозитории Maven недоступны, подготовьте локальный кэш зависимостей
   по инструкции в `docs/offline-build.md` и запустите приложение через
   вспомогательный скрипт:
   ```bash
   ./scripts/run-offline.sh maven-cache
   ```
5. Открыть в браузере: `http://localhost:8080` (будет редирект на страницу входа `/login`).

### Доступ администратора по умолчанию

- логин: `admin`  
- пароль: `admin`  

Роль администратора нужна для управления каталогом товаров, категориями и просмотра статистики.
