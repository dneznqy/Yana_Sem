# Информационно-справочная система интернет-магазина

Курсовой проект по дисциплине «Современные технологии программирования».

## Стек технологий

- Java 17  
- Spring Boot 3 (Web, Security, Data JPA, Validation, Thymeleaf)  
- PostgreSQL  
- Maven

## Как запустить

1. Установить PostgreSQL (если ещё не установлен). После установки в СУБД уже есть стандартная база данных `postgres`, она и будет использоваться приложением.
2. При необходимости изменить логин/пароль к БД в файле  
   `src/main/resources/application.properties`  
   По умолчанию используются:
   - пользователь: `postgres`
   - пароль: `postgres`
3. В корне проекта выполнить команду:
   ```bash
   mvn spring-boot:run
   ```
   При первом запуске Hibernate автоматически создаст все необходимые таблицы в базе `postgres`
   и инициализирует тестовые данные (администратор, категории, товары).
4. Открыть в браузере: `http://localhost:8080` (будет редирект на страницу входа `/login`).

### Доступ администратора по умолчанию

- логин: `admin`  
- пароль: `admin`  

Роль администратора нужна для управления каталогом товаров, категориями и просмотра статистики.
